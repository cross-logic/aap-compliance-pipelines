# Changelog

## 0.1.0 (2026-04-29)

### New Features
- `compliance_gather` module — agentless fact collection via SSH
- `compliance_evaluate` module — centralized STIG rule evaluation
- `normalize_xccdf` module — XCCDF to common findings format
- Scanner orchestration playbooks (install, scan, fetch, normalize, uninstall)
- 20 STIG rules defined as YAML data
- 85 unit tests + 3 integration test targets
